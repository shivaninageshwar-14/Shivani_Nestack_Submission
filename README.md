# Dream Portal QA Automation

## Candidate Details
Name: Shivani Howdekari

## Objective
Automated functional testing of the Dream Portal application using Playwright.

## Application Under Test
https://arjitnigam.github.io/myDreams/

## Test Scenarios Covered
- Login Validation
- Add Dream
- Edit Dream
- Delete Dream
- Form Validation
- UI Navigation Testing

## Tech Stack
- Playwright
- JavaScript
- Node.js

## Project Structure

tests/ → Automation test scripts  
screenshots/ → Execution screenshots  
videos/ → Test execution recordings  
playwright-report/ → HTML reports

## Installation

```bash
npm install
```

## Execute Tests

```bash
npx playwright test
```

## Generate Report

```bash
npx playwright show-report
```
